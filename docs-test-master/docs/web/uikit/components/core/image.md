## Overview
Responsive image component with variants


## Usage

```tsx live enableExportToCodePen enableSwitchTheme enableHideEditor
<Image
  src={"https://static.devfdg.net/static/mono-static/docs-ui/img/favicon.png"}
  sx={{
    width: [ '30%', '20%' ],
    borderRadius: 8,
  }}
/>
```


```tsx live enableExportToCodePen enableSwitchTheme enableHideEditor
<Image
  src={"https://static.devfdg.net/static/mono-static/docs-ui/img/favicon.png"}
  variant='avatar'
/>
```
